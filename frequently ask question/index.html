Frequently Ask Question

What is JavaScript ?

A high-level, interpreted programming language used for web development.

How is JavaScript different from Java ?

JavaScript is a scripting language for web pages, while Java is a standalone programming language.

What are JavaScript data types ?

String, Number, Boolean, Null, Undefined, Symbol, BigInt, Object, Array, Function.

What is a closure in JavaScript?

A function that retains access to its lexical scope even when invoked outside that scope.
